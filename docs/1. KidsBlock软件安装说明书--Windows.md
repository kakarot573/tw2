# 1.1TwinCode软件安装说明书--Windows
<span style="color: rgb(255, 76, 65);">注意：这里是以Windows系统为例</span>

1. 先找到twincode软件安装包，双击下载好的软件“twincode Desktop.exe”![Img](./media/TwinCode图标.png)。 

2. 先选中“**为使用这台电脑的任何人安装**”，再点击“**下一步**”。



3. 先点击“**浏览（B）...**”,选择安装的位置（<span style="color: rgb(255, 76, 65);">我这里选择安装在D盘，你也可以选择安装在电脑的其他盘，最好是不含中文的路径</span>），再点击“**安装**”。这样，软件就在安装过程中。

![Img](./media/Windows2.png)

![Img](./media/w3.png)

4. 几秒种后，安装完成。点击“**完成**”就可以打开安装好的软件。



5. 如果出现电脑安全警报窗口，点击“**允许访问**”。这样就可以打开了软件页面。



![Img](./media/w6.png)

## 1.2TwinCode软件使用的常见问题
<span style="color: rgb(255, 76, 65);">注意：这里是以Windows系统为例</span>

1. 当下载软件遇到“文件夹拒绝访问”而安装失败时，大多数是因为后台其它软件开的太多，关闭后台运行的软件重新安装即可。（安装目录最好是全英文，例如D:\APP\TwinCode）

2. 卸载时可以右键软件图标t，然后点击“打开文件所在的位置”。
![Img](./media/image-2.png)
选择Uninstall TwinCode.exe双击执行即可卸载软件。
![Img](./media/image-1.png)

## 1.3TwinCode软件的驱动安装方法和使用方法

1.软件中各按钮的功能：

![Img](./media/image3.png)

2.点击![Img](./media/image2.png)可以选择语言“**English**”或“**简体中文**”。


3.点击![Img](./media/ABCD4.png)，选择点击“**安装驱动**”，安装驱动。<span style="color: rgb(255, 76, 65);">（注意：如果电脑已经安装了驱动程序，则不需要再安装驱动；如果没有，则需要进行以下操作）</span>

![Img](./media/ABCD5.png)

A.在“**设备驱动程序安装向导**”页面选择点击“**下一页**”。

![Img](./media/ABCD6.png)

B.过一会儿，选择点击“**完成**”。

![Img](./media/ABCD7.png)

C.选择点击“**下一页**”。

![Img](./media/ABCD8.png)

D.选择点击“**完成**”。

![Img](./media/ABCD9.png)

E.如果出现安全页面，选择点击“**允许**”即可，然后选择点击“**Install**”。

F.选择点击“**下一页**”。

![Img](./media/ABCD15.png)

G.选择点击“**我接受这个协议**”后，点击“**下一页**”。

![Img](./media/ABCD16.png)

H.选择点击“**完成**”。

![Img](./media/ABCD17.png)

I.选择“**安装**”。

![Img](./media/ABCD18.png)

J.过几秒钟后，驱动安装完成，点击“**确定**”即可。

![Img](./media/ABCD19.png)

若出现安装失败，可能已经安装过该驱动，则不需要再安装。

![Img](./media/image-3.png)

![Img](./media/image4.png)

![Img](./media/image5.png)

![Img](./media/image6.png)

![Img](./media/image7.png)

![Img](./media/image-9.png)

6.Arduino Uno和端口（COM）都已经连接上了，点击![Img](./media/image8.png)，则![Img](./media/image8.png)会自动切换成![Img](./media/image9.png)。

7.左下角的![Img](./media/ABCD36.png)是添加传感器/模块和元件的扩展包，要想使用扩展需要先确认右上角处于“编程模式”。
![Img](./media/image9.png)

先点击![Img](./media/ABCD36.png)可以进入传感器/模块扩展库界面，然后点击传感器/模块，就可以添加对应的传感器/模块。例如需要添加无源蜂鸣器模块，点击“**无源蜂鸣器**”模块，“**未加载**”变成“**已加载**”，说明“**无源蜂鸣器**”模块![Img](./media/ABCD37.png)添加成功。

![Img](./media/ABCD38.png)

![Img](./media/image-10.png)

![Img](./media/image-11.png)

点击![Img](./media/ABCD41.png)，可以回到代码编辑器界面，在模块区可以看到添加的“**无源蜂鸣器**”指令方块。

![Img](./media/ABCD42.png)

如果需要删除“**无源蜂鸣器**”模块，只需要点击再次进入传感器/模块扩展库界面，点击“**无源蜂鸣器**”模块，“**已加载**”变成“**未加载**”，则说明“**无源蜂鸣器**”模块![Img](./media/ABCD43.png)删除成功。

![Img](./media/image-11.png)

![Img](./media/image-10.png)

其他的传感器/模块和元器件的添加和删除，方法是一样的。
